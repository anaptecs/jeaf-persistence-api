# JEAF Persistence API README #

This repository contains the source code of JEAF Persistence API. JEAF Persistence API simplifies usage of JPA. Therefore a so called persistence service provider is defined that can be used independent from the concrete environment. 

## Links ##
For further information please refer to:

* [JEAF Persistence](https://anaptecs.atlassian.net/l/c/E5zcD4xT)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.persistence/jeaf-persistence-service-provider-api)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.persistence/jeaf-persistence-service-provider-api/latest/index.html)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
